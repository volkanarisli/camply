Boxes for use in the terminal

The list of boxes is just a JSON file and can be used wherever.

Install

$ npm install --save cli-boxes

Usage

const cliBoxes = require('cli-boxes');
console.log(cliBoxes.single);
/*
{
"topLeft": "┌",
"topRight": "┐",
"bottomRight": "┘",
"bottomLeft": "└",
"vertical": "│",
"horizontal": "─"
}
*/
