Extend an object with the properties of additional objects. node.js/javascript util.

Install

Install with npm

$ npm i extend-shallow --save

Usage

var extend = require('extend-shallow');
extend({a: 'b'}, {c: 'd'})
//=> {a: 'b', c: 'd'}

Pass an empty object to shallow clone:

var obj = {};
extend(obj, {a: 'b'}, {c: 'd'})
//=> {a: 'b', c: 'd'}
